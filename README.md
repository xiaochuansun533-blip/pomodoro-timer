# 番茄钟 ⏰

一个用 **HTML + CSS + JavaScript** 实现的简易番茄钟（Pomodoro Timer）。

## 功能 ✨
- **25 分钟倒计时**：专注工作学习
- **5 分钟倒计时**：短暂休息
- 倒计时结束自动播放提示音
- 防止重复点击（按钮会在倒计时期间禁用）

## 使用方法 🚀
1. 打开页面后，点击 **开始** 按钮启动 25 分钟倒计时。
2. 点击 **休息** 按钮启动 5 分钟倒计时。
3. 等待倒计时结束，提示音响起后按钮会恢复。
